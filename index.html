Big Projects

1212!
2015

1212! is a simple little puzzle game where you place pieces, make lines, and score points, all the while being careful not to run out of room.

Basic Physics Thingy
2013 +

This is a basic physics sandbox I've been developing since 11th grade. I've occasionally updated it as I've learned more complex physics.

3D Music Analyzer (WebGL)
2014

Uses the WebAudio API and ThreeJS to visualize a audio stream in 3D realtime. Supports both local files and microphone input.

2D Music Analyzer (Canvas)
2014

Uses the WebAudio API and HTML5 Canvas to visualize a audio stream in 2D realtime. Supports both local files and microphone input.

Cellular Automa
2013

A basic Cellular Automa generator I wrote as a school project.

Conway's Game of Life
2013

A basic implementation of Conway's Game of Life in HTML5 canvas I wrote for a school project.
